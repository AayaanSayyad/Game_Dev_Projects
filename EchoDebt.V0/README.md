# Echo Debt

A minimal browser orbit-dodger. Collect nodes on the ring while avoiding pulses from the center.

## Play locally

```bash
cd echo-debt
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080)

## Controls

| Key | Action |
|-----|--------|
| `←` `→` or `Space` | Reverse orbit direction |
| `Enter` | Start / continue |

## v1 scope

- 5 levels with increasing difficulty
- 3 lives per run
- Keyboard-only (laptop/desktop)
- Code-drawn graphics + synthesized SFX

## Planned (v2+)

- Echo ghosts on hit (rewind mechanic)
- Relief power-ups (Redact, Stutter, etc.)
- Endless mode
